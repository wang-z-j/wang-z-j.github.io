    • 打开 crates.io。
    • 使用你的 GitHub 账户登录到 crates.io

  • 设置 cargo 的 API Token

    • 登录后，点击右上角的个人头像，选择 “Account Settings”。
    • 在 API Tokens 部分，点击 “New Token” 来生成一个新的 API Token

  • 复制生成的 Token。

  • 在发布之前，确保你的 crate 符合发布要求：

    • 检查 Cargo.toml

    • 关键字段：

      • **name**：crate 的名称，必须全局唯一（未被占用）。
      • **version**：crate 的版本，符合语义化版本规则（如 0.1.0）。
      • **authors**：作者信息，通常是你的名字和邮箱地址。
      • **license**：你的 crate 的开源许可证，如 MIT 或 Apache-2.0。
      • **description**：简短描述你的 crate 的用途。
      • **repository**：你的代码存储库地址（如 GitHub URL）。
      • keywords 和 **categories**：用于在 crates.io 上更好地分类和搜索你的 crate。
